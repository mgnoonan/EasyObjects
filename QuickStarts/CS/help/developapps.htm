Developing Applications with EasyObjects.NET

This topic describes how to develop applications using EasyObjects.NET. It first explains how to configure the DynamicQuery provider and incorporate it into your applications. Next, in Key Scenarios, it explains how to use EasyObjects.NET in particular scenarios, such as the different ways to query the database, adding and updating data, transaction management and handling null values. Lastly, in Details of Development Tasks, it gives more information about topics such as connection management, parameter handling, and handling exceptions.

EasyObjects.NET ships as source code and must be compiled before the QuickStarts and configuration tools can be used. To learn how to compile the source code, see Building the EasyObjects.NET project.
