Retrieving Multiple Rows as XML

An example of where you may want to use XML data is within a web service, or to save data to another target besides a database.

Typical Goals

In this scenario, the goal is to retrieve data from the database and convert it to an XML format.

Solution

Use the ToXml method provided by the EasyObject class to convert the data from the database to an XML string. Use the FromXml method to load the business object from an XML string (typically generated using ToXml).

Using ToXml

The following code shows how to use the ToXml method.

[C#]

Products prod = new Products();
prod.LoadAll();

txtDisplay.Text = prod.ToXml();

[VB]

 Dim prod As Products = New Products
prod.LoadAll()

txtDisplay.Text = prod.ToXml()

Usage Notes

The ToXml method does not respect the DataRowState of the EasyObject. If you need to maintain the DataRowState, use the Serialize method instead.
